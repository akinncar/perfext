# Spec: <project>
Instructions for project project, the name of project is `Perfext`, a tool to make perfect texts.

## Summary
The project is a browser extension, that works like gramarly suggesting improvements for your texts, while you type in text inputs, after some time (like 5s or what you suggest, like lodash feature), user see a highglight in text that was verified, user can hover and see a buble with the suggestion, and click it to apply or not.
This suggestion is made via IA lib usage, user can define the API key and the model/AI user in the browser extension configuration in modal (this is what you can use React to make the UI), the UI shouldn't be a big thing.

Instructions are explicited in `browser.md`, `lp.md`
